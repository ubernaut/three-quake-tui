Title                   : UltraViolence

Filename                : UltraV.bsp

Author                  : Adriano Lorenzini aka Escher
Email Address           : bib2002@iperbole.bologna.it
Description             : A small map based on an egyptian theme.
                          This level has been designed for a *very* intensive
                          one-o-one deathmatch, it's all you ever wanted to play
                          if you love the rocket jumping.

* Play Information *

Game                    : Quake Registered 1.01 or higher
Map Name                : UltraViolence
Single Player           : NO
Co-op                   : NO
Deathmatch 2-4          : 6 Deathmatch Starts
Difficulty Settings     : NO
Additional Info         : No exit, No escape. Anyway setting a time limit
                          it will start the DM4 map
